Initialise cipher by generating cipher parameters such that p is a lambda bit prime and q is an eta bit prime. The public modulus for arithmetic operations on ciphertexts is pq

Parameters
lambdaBit length of secret modulus p (security parameter)
etaBit length of secret prime q.

HE1Encrypter::HE1Encrypter ( int  n,
int  d,
int  rho 
)

Initialise cipher by generating cipher parameters so that p is large enough to accommodate a degree d multivariate polynomial computed over n rho%-bit integer inputs and q is large enough so that pq cannot be factored in polynomial time.

Parameters
ntotal number of inputs
ddegree of polynomial to compute
rhobit length of inputs, e.g for 32-bit integers rho = 32
