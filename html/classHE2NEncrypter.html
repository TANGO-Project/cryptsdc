HE2NEncrypter::HE2NEncrypter ( int  lambda,
int  eta,
int  nu 
)

Initialise cipher by generating cipher parameters such that p is a lambda bit prime, q is an eta bit prime, and kappa is a nu bit prime. The public modulus for arithmetic operations on ciphertexts is pq

Parameters
lambdaBit length of secret modulus p (security parameter)
etaBit length of secret prime q
nuBit length of secret prime kappa.

HE2NEncrypter::HE2NEncrypter ( int  n,
int  d,
int  rho,
int  rhoprime 
)

Initialise cipher by generating cipher parameters so that p and kappa are large enough to accommodate a degree d multivariate polynomial computed over n rhoprime%-bit integers and q is large enough so that pq cannot be factored in polynomial time.

Parameters
ntotal number of inputs
ddegree of polynomial to compute
rhobit length of inputs, e.g for 32-bit integers rho = 32
rhoprimeThe effective entropy (bit lengths) of the inputs after encryption.
