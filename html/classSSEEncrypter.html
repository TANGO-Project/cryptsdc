◆ encrypt()

std::string SSEEncrypter::encrypt ( std::string &  plaintext)
overridevirtual

Encrypts the plaintext word as a triple of a 16-byte intialisation vector, an AES ciphertext, and a MAC tag. Currently, this triple is implmented as a hex encoded JSON object for convenience. However, it could easily be altered to return a C++ tuple or a concatenated byte sequence.

Parameters
plaintextA string
Returns
A JSON object
